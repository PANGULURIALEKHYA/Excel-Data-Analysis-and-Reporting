# Excel-Data-Analysis-and-Reporting

## Overview
This project demonstrates the use of Microsoft Excel for data cleaning, analysis, reporting, and dashboard creation using employee workforcr data.

## Skills Used
- Data Cleaning and Validation
- Pivot Tables
- VLOOKUP
- Advanced Excel Formulas
- Data Analysis
- Reporting
- Dashboard Creation
- Data Visualisation

## Analysis Performed
- Employee Count Analysis
- Salary Analysis
- Employees with Salary Greater Than 100k
- Highest and Lowest Salary Analysis
- Department-wise Employee Distribution
- Employee Lookup Functionality
- Gender Ratio Analysis
- Workforce Reporting

## Tools Used
- Microsoft Excel
- Pivot Tables
- VLOOKUP
- Conditional Formatting
- Excel Formulas

## Outcome
The project transformed raw employee data into structured reports and insights to support bussiness reporting and decision-making.
